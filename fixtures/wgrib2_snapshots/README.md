# wgrib2 Compatibility Snapshots

This directory stores golden outputs generated by `wgrib2` for CLI compatibility tests.

## Policy

- `wgrib2` is used only to generate snapshots locally.
- CI must not execute `wgrib2`.
- Tests compare our CLI output against these snapshot files.

## Update

From repository root:

```bash
bash tools/update_wgrib2_snapshots.sh
```

Optional custom binary path:

```bash
WGRIB2_BIN=/opt/homebrew/bin/wgrib2 bash tools/update_wgrib2_snapshots.sh
```
